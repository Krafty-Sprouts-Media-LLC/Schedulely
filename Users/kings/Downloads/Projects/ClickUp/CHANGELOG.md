# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 14/11/2024

### Changed
- Completely restructured December for animalofthings.MD file with extreme keyword diversity
- Reduced green snake concentration from 46 tasks to 10 tasks only
- Reduced cat facts from 40+ tasks to 15 tasks (1 batch only)
- Reduced frogs from unlimited to 30 tasks maximum (2 batches)
- Implemented strict ONE-BATCH-PER-KEYWORD-TYPE policy to eliminate repetition
- Reorganized all tasks by similar keywords grouping:
  - All red birds grouped together
  - All white birds grouped together
  - All brown birds grouped together
  - All frog types by location grouped together
  - All shark species grouped together
  - All whale/dolphin species grouped together
  - All caterpillars/spiders/beetles grouped together
  - All lizards/turtles/reptiles grouped together
  - All predator food chains grouped together
  - All flora-eating animals grouped together

### Added
- New diverse categories: Sharks (10 tasks), Whales & Dolphins (10 tasks)
- Reptile species (non-snake): Lizards, Turtles, Alligators, Iguanas (10 tasks)
- Amphibians & Jellyfish predators (10 tasks)
- Miscellaneous wildlife and adaptation facts (12 tasks)
- Enhanced bird category with specific color grouping (red/white/brown)

### Removed
- Excessive repetition of green snake geographic locations (reduced from 46 to 10)
- Over-concentration on cat facts (reduced from 40+ to 15)
- Mixed keyword batches - now each batch has ONE clear theme
